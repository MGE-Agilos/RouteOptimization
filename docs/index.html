Fonctionnalités
🗺️

Réseau OSM réel

3 039 segments routiers chargés depuis OpenStreetMap. Données à jour sur toute la commune d'OLLN.

📊

Modèle de trafic hybride

Simulation OD pondérée par 4 114 points d'intérêt + betweenness centrality ancrée sur les AADT belges.

🔄

Redistribution réelle

Quand une route est bloquée, le trafic se redistribue automatiquement sur les itinéraires alternatifs.

⚡

Simulation instantanée

Posez des panneaux sur n'importe quel segment et voyez l'impact sur tout le réseau en quelques secondes.
